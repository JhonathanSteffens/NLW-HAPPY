# NLW-HAPPY

Este é um website desenvolvido durante a Next Level Week, um projeto promovido pela RocketSeat que visa introduzir novas pessoas para o mundo da programação e ensinar conceitos de HTML, CSS e JavaScript. O website tratasse de uma plataforma onde pessoas podem encontrar orfanatos em sua cidade e agendar visitas, além de orfanatos poderem se cadastrar para serem encontrados
